# Traffic-Access-Dashboard 

*A simple MERN app created with Vite and MongoDB Atlas Atlas*

See the card of your favorite characters and their status. the database contains only a small part of the Rick and Morty API data, as it required the use of the MERN stack.
Also, the database used is read-only, so the registry cannot be modified.
The frontend of this app works on the [localhost:8000](http://localhost:8000/) and the backend runs on [localhost:5000](http://localhost:5000/).

![Earth C-137](https://raw.githubusercontent.com/metalpoch/rickAndMorty-MERN/main/EarthC-137.png)

## Installation

##### Clone this repository [GitHub](https://github.com/metalpoch/rickAndMorty-MERN#) 
```bash
git clone https://github.com/metalpoch/rickAndMorty-MERN.git
```

##### Use [npm](https://www.npmjs.com/) to install the dependencies 
```bash
cd rickAndMorty-MERN
npm install
cd server
npm install
cd ../client
npm install
cd ../  # to rickAndMorty-MERN directory
```

##### From the rickAndMorty-MERN directory execute:
```bash
npm run dev
```
